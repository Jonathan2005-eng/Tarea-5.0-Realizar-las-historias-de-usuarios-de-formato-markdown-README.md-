7. Ejercicio en clase
Transferencias rápidas
Como cliente del banco quiero realizar transferencias a otros usuarios desde mi celular para enviar dinero de forma rápida y segura.
Criterios de aceptación:
- El formulario debe permitir ingresar el número de cuenta, monto y concepto.
- La transferencia debe confirmarse con un código de verificación (OTP).
- El usuario debe recibir una notificación de éxito o error inmediatamente.

Consulta de movimientos
Como usuario de la app quiero revisar el historial de mis transacciones para tener control sobre mis gastos y depósitos.
Criterios de aceptación:
- Debe mostrar una lista con fecha, tipo de transacción, monto y saldo resultante.
- El usuario puede filtrar por rango de fechas y tipo de operación.
- La información debe actualizarse en tiempo real.

Seguridad de acceso
Como cliente quiero iniciar sesión con autenticación biométrica para proteger mi cuenta sin tener que recordar contraseñas.
Criterios de aceptación:
- La app debe permitir configurar huella digital o reconocimiento facial.
- El acceso debe bloquearse tras 3 intentos fallidos y mostrar un mensaje de advertencia.
- El usuario puede desactivar o cambiar el método de autenticación desde el perfil.
9. Actividad final
Programar mensajes
Como usuario de WhatsApp quiero poder programar el envío de mensajes para comunicarme en el momento adecuado sin tener que estar conectado.
Criterios de aceptación:
- El usuario puede seleccionar fecha y hora para enviar un mensaje.
- El mensaje programado se puede editar o cancelar antes del envío.
- Se muestra una confirmación cuando el mensaje ha sido enviado automáticamente.

Historia de usuario 2: Como usuario,
quiero que WhatsApp sugiera chats para archivar o eliminar para mantener mi bandeja organizada sin perder información importante.
Criterios de aceptación:
- La app identifica chats inactivos por más de 6 meses.
- El usuario recibe una notificación con sugerencias de limpieza.
- Se puede revisar y aprobar cada sugerencia antes de ejecutar la acción.

Historia de usuario 3: Búsqueda avanzada de archivos
Como usuario,quiero buscar documentos imágenes o audios por tipo y fecha para encontrar fácilmente archivos compartidos en mis conversaciones.
Criterios de aceptación:
- La búsqueda permite filtrar por tipo de archivo (imagen, documento, audio).
- Se puede establecer un rango de fechas para acotar los resultados.
- Los resultados muestran el nombre del archivo, fecha y chat de origen.
